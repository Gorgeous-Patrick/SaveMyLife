Rescue the system

  • Properly mount the zfs dataset onto the system. Run only in Live CD, or run curl https://rescue.baichuan.li/scripts/livecd_prep.sh | sh
  • Unmount the zfs dataset from the system. Run only in Live CD, or run curl https://rescue.baichuan.li/scripts/livecd_umount.sh | sh
  • Command to rebuild initramfs: sudo mkinitcpio -P
  • Location of the grub parameter: /etc/default/grub
  • Command to rebuild grub: sudo grub-mkconfig -o /boot/grub/grub.cfg
